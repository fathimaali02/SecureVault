4️⃣  STEGANOGRAPHY (LSB)
   ┌─────────────────────────────────────────────────────┐
   │ Normal Image:  RGB pixel = (R=255, G=100, B=50)     │
   │                                                       │
   │ With 1 data bit embedded:                            │
   │   R LSB = 1: (255 → 255)  [visually identical]      │
   │   G LSB = 0: (100 → 100)  [imperceptible change]   │
   │   B LSB = 1: (50 → 51)    [invisible to human eye]  │
   │                                                       │
   │ Capacity: 1 bit per RGB component                    │
   │ = 3 bits per pixel × (800×600) = ~1.4 MB per image │
   │                                                       │
   │ Detection: Requires LSB analysis tools + password    │
   └─────────────────────────────────────────────────────┘
   
   Security: Stealth + encryption = double protection

5️⃣  BLOCKCHAIN LEDGER
   ┌─────────────────────────────────────────────────────┐
   │ Genesis Block:  index=0, hash=sha256('GENESIS')     │
   │                                                       │
   │ File Block:                                          │
   │   {                                                   │
   │     "index": 1,                                      │
   │     "timestamp": "2025-01-08T...",                  │
   │     "file_id": "a1b2c3d4...",                       │
   │     "file_hash": "sha256(encrypted_file)",          │
   │     "ipfs_cid": "QmXxxx...",                        │
   │     "prev_hash": "sha256(genesis_block)"            │
   │   }                                                   │
   │                                                       │
   │ Verification:                                        │
   │   - Each block points to previous via hash           │
   │   - Breaking one block invalidates entire chain      │
   │   - File modification = hash mismatch = detected     │
   └─────────────────────────────────────────────────────┘
   
   Security: Immutable audit trail
